# 우리이야기 - Cafe24 연동 워크플로우 계획

## 📋 프로젝트 개요

**프로젝트명**: 우리이야기 (당신의 이야기를 옷에 담다)
**목표**: HTML/CSS 기반 커스텀 쇼핑몰 → Cafe24 통합 쇼핑몰로 전환
**기간**: 8-10주 (예상)

---

## 🎯 Phase별 상세 계획

### **Phase 1️⃣: Cafe24 기본 설정** (1-2주)

#### 1.1 Cafe24 쇼핑몰 생성
- [ ] Cafe24 가입 (cafe24.com)
- [ ] 쇼핑몰 생성 및 기본 설정
- [ ] 사업자정보 등록
  - 사업자번호
  - 사업장 주소
  - 대표자명 & 연락처

#### 1.2 도메인 설정
- [ ] 도메인 구입 또는 연결
  - 추천: uriiyagi.com 또는 uriiyagi.kr
  - Cafe24 도메인 연결
- [ ] SSL 인증서 설정 (https)

#### 1.3 기본 설정
- [ ] 환불정책 작성
- [ ] 배송정책 설정
  - 배송료 (기본배송비: 3,000원, 제주/도서산간 +5,000원)
  - 배송기간 (3-5일)
- [ ] 개인정보처리방침 & 이용약관 작성

---

### **Phase 2️⃣: 상품 등록** (1주)

#### 2.1 카테고리 구조
- 아동복 (3개 상품: 반팔, 긴팔, 커플)
- 성인복 (3개 상품: 반팔, 긴팔, 친구 커플)
- 시니어 (3개 상품: 여성, 남성, 긴팔)
- 반려동물복 (3개 상품: 강아지, 고양이, 커플)
- 단체복 (3개 상품: 어린이집, 회사, 스포츠팀)

#### 2.2 상품 등록 정보
- 상품명, 가격(29,900~39,900원), 설명, 이미지(3-5장)
- 옵션: 색상(5가지), 사이즈(5가지)

#### 2.3 커스터마이징 옵션
- 아동복: 이름 + 출생년도 → 띠&별자리 자동계산
- 성인복: 감성 문구 선택 (10개)
- 시니어: 추억 명언 선택 (24개)
- 반려동물: 이름 + 품종 + 생일
- 단체복: 팀명 + 사훈 + 유형

---

### **Phase 3️⃣: 디자인 & 홈페이지** (2주)

#### 3.1 Cafe24 테마 커스터마이징
- 색상: Gold(#c9a84c), Ink(#1a1a2e), Paper(#faf8f3)
- 폰트: Nanum Brush Script, Noto Serif KR

#### 3.2 페이지 구성
**메인**: 헤더 + 영웅섹션 + 5개 카테고리 카드 + 푸터
**카테고리**: 카테고리명 + 상품 그리드
**상품 상세**: 이미지 + 설명 + 옵션 + **커스터마이징 폼** + 미리보기 + 구매버튼

---

### **Phase 4️⃣: 커스터마이징 기능** (2-3주)

#### 4.1 입력 폼 설계
```
아동복: 아이이름 + 출생년도 → 자동계산
성인복: 감성문구 dropdown
시니어: 추억명언 dropdown
반려동물: 이름 + 타입 + 품종 + 생일
단체복: 팀명 + 이름 + 유형 + 사훈
```

#### 4.2 미리보기 기능
- 입력값 실시간 SVG 반영
- 색상/사이즈 선택 시 즉시 업데이트

#### 4.3 구현 방법
- Option A: Cafe24 앱 개발 (권장, 안정성)
- Option B: HTML/JS 임베드 (빠른 구현)

---

### **Phase 5️⃣: 결제 & 배송 시스템** (1-2주)

#### 5.1 결제 게이트웨이
- PG사: 나이스페이, 이니시스, KCP 중 선택
- 신용카드, 계좌이체, 휴대폰 결제

#### 5.2 배송 연동
- 배송사: CJ, 로젠, 우체국
- 배송비: 기본 3,000원 (제주/산간 +5,000원, 50,000원 이상 무료)
- 자동 배송 추적

---

### **Phase 6️⃣: 마케팅 & SEO** (1주)

#### 6.1 SEO 최적화
- 메타 태그, 상품 키워드, Schema.org
- Google Search Console & Naver 웹마스터

#### 6.2 SNS 연동
- Instagram, Facebook 연동
- 상품 공유 버튼

---

### **Phase 7️⃣: 통합 테스트 & 배포** (1주)

#### 7.1 기능 테스트
- 회원가입/로그인, 상품검색, 장바구니, 커스터마이징

#### 7.2 결제 & 배송 테스트
- 모든 결제 수단 테스트
- 배송 자동화 확인

#### 7.3 최종 점검
- SSL 인증, 속도 최적화, 모바일 반응형
- 라이브 배포 🚀
